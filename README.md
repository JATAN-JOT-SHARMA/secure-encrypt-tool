# 🔐 Secure Encrypt Tool

![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)
![Status](https://img.shields.io/badge/Status-Active-success.svg)

Secure Encrypt Tool is a Python-based desktop application that helps users protect sensitive files using strong encryption. It provides an easy-to-use graphical interface for encrypting and decrypting files while maintaining data confidentiality and integrity.

> **Note:** This project is intended for educational purposes and authorized use only.

---

# 📌 Features

## 🔒 File Encryption
- Encrypt files with strong encryption algorithms
- Password-protected encryption
- Supports multiple file formats
- Fast encryption process

## 🔓 File Decryption
- Decrypt previously encrypted files
- Password verification
- Restore original file contents
- Prevent unauthorized access

## 🗂️ Batch Processing
- Encrypt multiple files at once
- Decrypt multiple files
- Folder support
- Progress tracking

## 🔑 Password Security
- Secure password input
- Password strength validation
- Optional password visibility toggle

## 📄 File Information
- File name
- File size
- Last modified date
- Encryption status

## 📊 Logging
- Encryption history
- Decryption history
- Error logging
- Activity records

## 🎨 Modern GUI
- Clean dark theme
- Responsive interface
- Drag-and-drop support (optional)
- Progress bars and status updates

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/YourUsername/Secure-Encrypt-Tool.git
```

```bash
cd Secure-Encrypt-Tool
```

---

## Create a Virtual Environment

```bash
python -m venv venv
```

Activate it:

**Windows**

```bash
venv\Scripts\activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run the Application

```bash
python main.py
```
